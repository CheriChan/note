## 节流

每隔一段时间执行一次

原理：设置一个定时器，约定x毫秒后执行，每隔一段时间执行一次函数，且重置定时器

```js
function throttle(func, wait) {
    let timeout = null
    return function() {
        let context = this
        let args = arguments
        if (!timeout) {
            timeout = setTimeout(() => {
                timeout = null
                func.apply(context, args)
            }, wait)
        }

    }
}

```
